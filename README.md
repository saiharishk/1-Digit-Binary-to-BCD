# 4 Bit Binary to BCD

This verilog code converts a 4 bit binary number to BCD  

Example: 1111 (15 in decimal) -- 1 0101, where 1 represents the tens place and 0101 represents the units place.  

Sample output:  
4:::0 : 0100  
1:::0 : 0001  
9:::0 : 1001  
3:::0 : 0011  
13:::1 : 0011  
12:::1 : 0010
